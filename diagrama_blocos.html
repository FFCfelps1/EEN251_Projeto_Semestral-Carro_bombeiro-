Estrutura do Pacote RF

4 bytes · payload estático · NRF24L01 Enhanced ShockBurst · ~12 pacotes/s
Byte Campo Tipo Faixa Descrição
0 vel_esq int8 −100 … +100 Velocidade lado esquerdo (negativo = ré)
1 vel_dir int8 −100 … +100 Velocidade lado direito (negativo = ré)
2 bomba uint8 0 / 1 0 = OFF · 1 = Bomba d'água ligada
3 stop uint8 0 / 1 0 = operação normal · 1 = parada de emergência
Mapeamento Tank Drive — Joystick → Velocidade Diferencial
vel_esq = Y + X
vel_dir = Y − X
// Y = eixo vertical (avanço/recuo), X = eixo horizontal (giro)
// Valores limitados ao intervalo [−100, +100] antes do envio

Diagrama de Alimentação

7,4V
Bateria 2S 18650
~2000 mAh · corrente pico ~4A
→ DC-DC
Regulador MP1584
Buck converter · eficiência >90%
5V
Saída regulada
L298N, Relé, Pico VSYS
3,3V
Saída interna Pico
NRF24L01, sensores, GPIOs
Distribuição de consumidores por tensão
7,4V direto →  L298N (pino VS) — alimentação dos motores
5V regulado →   L298N (VCC lógico) · Relé 5V (bobina) · Pico (VSYS)
3,3V (Pico) →  NRF24L01 VCC · Sensor de nível · LEDs · OLED
⚠ Atenção →   NRF24L01 opera em 3,3V — nunca conectar ao pino 5V
